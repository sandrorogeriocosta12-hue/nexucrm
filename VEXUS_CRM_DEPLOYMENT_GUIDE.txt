🎯 3 PASSOS PARA FAZER DEPLOYMENT:

1️⃣  CRIAR REPOSITÓRIO NO GITHUB
    ────────────────────────────────────────────────────
    Passo 1: Acesse https://github.com/new
    Passo 2: Preencha:
             • Repository name: vexus-crm
             • Description: CRM Inteligente com IA
             • Public (para Railway ver)
    Passo 3: Create repository
    
    ✨ Copie a URL que aparece (tipo: https://github.com/seu-usuario/vexus-crm.git)

2️⃣  FAZER PUSH DO CÓDIGO
    ────────────────────────────────────────────────────
    Cole estes comandos no terminal:
    
    cd /home/victor-emanuel/PycharmProjects/Vexus\ Service/vexus_crm_deployment
    
    git remote add origin https://github.com/SEU_USUARIO/vexus-crm.git
    git branch -M main
    git push -u origin main
    
    ✓ Pode demorar alguns segundos (1-2 MB de dados)

3️⃣  FAZER DEPLOY AUTOMÁTICO NO RAILWAY
    ────────────────────────────────────────────────────
    
    OPÇÃO A (Interface Web - Mais Fácil):
    
    1. Acesse https://railway.app
    2. Clique "New Project"
    3. Selecione "Deploy from GitHub"
    4. Autorize o Railway acessar seu GitHub
    5. Selecione o repositório "vexus-crm"
    6. Clique "Deploy"
    7. Railway faz tudo automaticamente!
    
    OPÇÃO B (Railway CLI - Mais Rápido):
    
    npm i -g @railway/cli
    cd /path/to/vexus-crm
    railway login
    railway init
    railway up
