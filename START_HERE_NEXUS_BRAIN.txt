ALGORITHM AT A GLANCE:

INPUT:  E (engagement), I (intention), U (ai_confidence) ∈ [0, 1]

PROCESS:
  1. Fuzzificación:  Raw values → Membership degrees [0, 1]
  2. Inference:      12 fuzzy rules → Activations
  3. Defuzzification: Weighted centroid → Score [0, 1]
  4. P_f:            Sigmoid(score) → Probability [0, 1]

OUTPUT: Score, P_f, Temperature (cold/warm/hot), Recommendation

LEARNING (Automatic):
  Error = actual_outcome - predicted_score
  w_i := w_i - η * error * μ_i  [η = 0.05]
  → Weights adjust automatically
  → System learns from feedback
