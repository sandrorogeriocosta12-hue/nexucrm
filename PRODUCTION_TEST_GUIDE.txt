════════════════════════════════════════════════════════════════════════════════
  🔧 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════════

❌ PROBLEMA: "Conexão recusada" ao acessar http://200.143.198.66:8000
   ✅ SOLUÇÃO: 
      1. Ver se servidor está rodando: ps aux | grep uvicorn
      2. Ver logs: tail -f logs/app.log
      3. Reiniciar: pkill uvicorn && python -m uvicorn app.api_main:app --host 0.0.0.0 --port 8000

❌ PROBLEMA: Firewall bloqueando
   ✅ SOLUÇÃO: sudo ufw allow 8000

❌ PROBLEMA: Erro de login "Invalid credentials"
   ✅ SOLUÇÃO: Verificar se contas estão no banco: psql -U vexus vexus_db -c "SELECT * FROM users;"

❌ PROBLEMA: Latência/lentidão
   ✅ SOLUÇÃO: 
      1. Ver CPU: top
      2. Ver memória: free -h
      3. Ver PostgreSQL: systemctl status postgresql
