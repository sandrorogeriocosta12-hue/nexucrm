                            🚀 GUIA RÁPIDO (5 MINUTOS)
════════════════════════════════════════════════════════════════════════════════

PASSO 1: Criar App WhatsApp
────────────────────────────────────────────────────────────────────────────────
   1. Acesse https://developers.facebook.com
   2. Clique "Meus Aplicativos" → "Criar Aplicativo"
   3. Tipo: "Business"
   4. Adicione produto: "WhatsApp"
   ⏱️ Tempo: ~2 minutos

PASSO 2: Obter Credenciais (3 coisas importantes)
────────────────────────────────────────────────────────────────────────────────
   
   Onde encontrar cada uma:
   
   PHONE_NUMBER_ID:
   └─ Meta Business Suite → WhatsApp Settings
   └─ Copia do número seu comercial (só números)
   
   BUSINESS_ACCOUNT_ID:
   └─ Meta Business Suite → Settings → Business Info
   └─ ID único da empresa
   
   ACCESS_TOKEN:
   └─ App Seu → Settings → User Token
   └─ Gere um novo → Permissões: whatsapp_business_*
   └─ NUNCA compartilhe este token!
   
   ⏱️ Tempo: ~2 minutos

PASSO 3: Configurar no Nexus CRM
────────────────────────────────────────────────────────────────────────────────
   1. Abra http://localhost:8000/frontend/app.html
   2. Login com sua conta
   3. Clique ⚙️ (Configurações) → WhatsApp Business API
   4. Preencha as 3 credenciais acima
   5. Clique "Salvar Configuração"
   6. Clique "Testar Conexão" ✅
   ⏱️ Tempo: ~1 minuto

PRONTO! Seu WhatsApp está integrado! 🎉

    └─ Escalação humana quando necessário
    
    ↓ retorna resposta

════════════════════════════════════════════════════════════════════════════════
                            🔌 ENDPOINTS DISPONÍVEIS
════════════════════════════════════════════════════════════════════════════════

GET /api/whatsapp/config
   └─ Retorna configuração atual (sem token por segurança)

POST /api/whatsapp/config
   └─ Salva novas credenciais
   └─ Body: { "phone_number_id", "business_account_id", "access_token", "webhook_verify_token" }

POST /api/whatsapp/send
   └─ Envia mensagem
   └─ Body: { "to": "5511999999999", "message": "Texto", "message_type": "text" }

POST /api/whatsapp/webhook
   └─ Recebe mensagens E eventos do WhatsApp
   └─ Configure em: Meta Developers → Apps → WhatsApp → Webhooks

GET /api/whatsapp/webhook (verification)
   └─ Valida webhook durante setup no Meta

POST /api/whatsapp/test
   └─ Testa conexão com WhatsApp

POST /api/whatsapp/template/list
   └─ Lista templates pre-aprovados

GET /api/whatsapp/status
   └─ Status atual da integração

                         🧪 EXEMPLOS DE USO (Terminal)

Testar API via CURL:

# 1. Salvar Configuração
curl -X POST http://localhost:8000/api/whatsapp/config \
  -H "Content-Type: application/json" \
  -d '{
    "phone_number_id": "123456789",
    "business_account_id": "987654321",
    "access_token": "EAA...seu_token",
    "webhook_verify_token": "seu_webhook_token"
  }'

# 2. Testar Conexão
curl -X POST http://localhost:8000/api/whatsapp/test

# 3. Enviar Mensagem
curl -X POST http://localhost:8000/api/whatsapp/send \
  -H "Content-Type: application/json" \
  -d '{
    "to": "5511999999999",
    "message": "Olá! Teste do Nexus CRM",
    "message_type": "text"
  }'

# 4. Listar Templates
curl -X POST http://localhost:8000/api/whatsapp/template/list
