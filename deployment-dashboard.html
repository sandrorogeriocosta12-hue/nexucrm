🔧 Guia de Configuração

1 Infraestrutura Cloud Pendente

Objetivo

Criar e configurar infraestrutura na nuvem (DigitalOcean recomendado)

Passos

  • 1. Acesse digitalocean.com e crie uma conta
  • 2. Crie um novo App Platform Project
  • 3. Configure PostgreSQL managed database
  • 4. Configure Redis managed cache
  • 5. Conecte seu repositório GitHub

Tempo Estimado

30-45 minutos

Recurso de Exemplo

DigitalOcean App Platform
- Suporta deploy automático via GitHub
- PostgreSQL 15 gerenciado
- Redis 7 gerenciado
- Custo: ~$150/mês para MVP
2 Gerar Chaves de API Pendente

Chaves Necessárias

🤖 OpenAI API Key

  • 1. Acesse platform.openai.com
  • 2. Faça login com sua conta
  • 3. Vá em API keys → Create new secret key
  • 4. Copie a chave sk-proj-xxxxx

📱 WhatsApp Business API Key

  • 1. Acesse business.facebook.com
  • 2. Crie uma aplicação WhatsApp Business
  • 3. Gere o token de acesso
  • 4. Obtenha o Phone Number ID

Tempo Estimado

20-30 minutos

⚠️ Importante: Guarde estas chaves em local seguro. Não compartilhe com ninguém!
3 Configurar Variáveis de Ambiente Pendente

Arquivo: .env.production

Copie o arquivo .env.production.example para .env.production e preencha com seus dados:

Variáveis Essenciais

FLASK_ENV=production OPENAI_API_KEY=sk-proj-xxxxx WHATSAPP_BUSINESS_API_KEY=xxxxx WHATSAPP_PHONE_NUMBER_ID=xxxxx DATABASE_URL=postgresql://user:pass@host:5432/dbname REDIS_URL=redis://host:6379/0 JWT_SECRET_KEY=xxxxx

4 Deploy em Staging Pendente

Objetivo

Fazer deploy do sistema em ambiente de teste antes de produção

Passos

  • 1. Faça push do seu código para GitHub
  • 2. DigitalOcean detectará e fará deploy automático
  • 3. Acesse a URL de staging fornecida
  • 4. Teste todos os endpoints via Swagger: /docs
  • 5. Valide as integrações (WhatsApp, OpenAI, etc)

Comandos

# Fazer push para GitHub git add . git commit -m "Config: Deploy em staging" git push origin main # Acessar Swagger https://seu-app-staging.ondigitalocean.app/docs

Tempo Estimado

2 horas (inclui testes)

Validações Necessárias

✅ API respondendo (GET /health)
✅ Banco de dados conectado
✅ Redis funcionando
✅ Endpoints de CRUD funcionando
✅ Autenticação JWT funcionando
5 Deploy em Produção Pendente

Objetivo

Colocar o sistema em produção com estratégia Blue-Green

Estratégia Blue-Green

  • 1. Ambiente Blue (atual) continua ativo
  • 2. Ambiente Green (novo) recebe deploy
  • 3. Testa tudo no Green
  • 4. Direciona tráfego para Green
  • 5. Blue permanece como backup

Comandos

# Em DigitalOcean, configure variáveis de ambiente de produção # Depois faça push de uma tag de versão git tag -a v1.0.0 -m "Production release" git push origin v1.0.0

Tempo Estimado

1-2 horas (com monitoramento)

⚠️ Precauções:
- Faça backup do banco de dados
- Monitore logs por 1 hora
- Tenha plano de rollback pronto
- Teste críticos endpoints após deploy

📚 Bem-vindo ao Vexus CRM!

O que é Vexus CRM?

Vexus CRM é um sistema de gerenciamento de relacionamento com clientes (CRM) inteligente, alimentado por IA artificial, que automatiza todo o processo de vendas.

Por que você precisa?

  • Automatização Completa: A IA trabalha 24/7 qualificando leads, respondendo mensagens e fechando vendas
  • Aumento de Conversão: Responde em segundos com mensagens personalizadas e persuasivas
  • Economia de Tempo: Seu time se foca em relacionamento, IA faz o resto
  • Omnichannel: Funciona em WhatsApp, Email, Instagram, SMS, Telegram, Facebook e Website simultaneamente
  • Inteligência em Tempo Real: Dashboards mostram exatamente o que está acontecendo agora

⚙️ Como Funciona na Prática

Fluxo Completo do Vexus CRM:

1️⃣ Cliente Chega (Qualquer Canal)
Cliente envia mensagem no WhatsApp, email, IG ou website
2️⃣ IA Recebe & Analisa
Sistema capta mensagem, analisa intenção e sentimento
3️⃣ IA Classifica Lead
Lead Scoring Agent classifica: Hot, Warm ou Cold
4️⃣ IA Responde
IA gera resposta personalizada em segundos (com sua voz)
5️⃣ Sistema Qualifica
Faz perguntas qualificadoras (BANT: Budget, Authority, Need, Timeline)
6️⃣ Cria ou Move Lead
Cria novo contato ou move no pipeline (Lead → Qualified → Negotiating...)
7️⃣ Agenda Follow-up
IA agenda automaticamente próximo contato (email, SMS, chat)
8️⃣ Você Recebe Notificação
You get notified only if it needs human touch (escalation)
9️⃣ Fechamento
IA detecta sinais de compra e faz proposta final
🔟 Pós-Venda
Sistema acompanha cliente, recomenda upsells
🤖 Os 7 AI Agents Inteligentes

1. Lead Scoring Agent

O que faz: Classifica automaticamente a qualidade de cada lead

Como funciona: Analisa comportamento, engajamento, respostas e cria score de 0-100

Resultado: Você sabe exatamente quem está pronto para comprar

2. Conversation Analyzer Agent

O que faz: Entende o que o cliente REALMENTE quer dizer

Como funciona: Detecta intenção (pergunta, objeção, compra), sentimento (feliz, frustrado), urgência

Resultado: IA responde com empátia e relevância

3. Next Best Action Agent

O que faz: Sugere a melhor ação para cada situação

Como funciona: "Esse lead é Hot e tem objeção de preço → ofereça desconto" ou "Esse cliente já comprou 5x → faça upsell"

Resultado: Aumenta conversão em 40-60%

4. Proposal Generator Agent

O que faz: Cria propostas personalizadas automaticamente

Como funciona: Pega dados do cliente (tamanho, industria, budget) e gera proposta customizada em PDF

Resultado: Propostas em minutos, não em horas

5. Followup Scheduler Agent

O que faz: Agenda follow-ups no momento exato

Como funciona: Aprende melhor hora para responder cada lead, agenda contato automático

Resultado: 100% dos leads recebem follow-up (humanos esquecem, IA não)

6. Channel Optimizer Agent

O que faz: Escolhe o melhor canal para contatar cada cliente

Como funciona: "João responde melhor no WhatsApp → mande pelo WA", "Maria usa email → mande email"

Resultado: Respostas 3x mais rápidas

7. Pipeline Manager Agent

O que faz: Move leads automaticamente no pipeline

Como funciona: "Lead qualificado com orçamento aprovado → move para Negotiating"

Resultado: Pipeline sempre atualizado e em tempo real

📊 Como Usar o Visual Pipeline

O que é?

Um quadro visual (tipo Pipefy) que mostra ONDE está cada lead no processo de vendas

As 6 Fases Padrão:

1. Lead
Primeiro contato, sem qualificação ainda
2. Qualified
Respondeu perguntas, tem interesse
3. Negotiating
Recebeu proposta, discutindo preço
4. Review
Analisando proposta, quase fechando
5. Won
✅ Vendido! Começar relacionamento
6. Lost
❌ Perdido. Manter contato para futuro

Como Funciona na Prática:

  • Cards: Cada lead é um "card" (cartão) no quadro
  • Arrastar: Você pode arrastar cards entre fases (ou IA faz automático)
  • Informações: Cada card mostra: nome, empresa, score, último contato, próxima ação
  • Filtros: Veja só leads "Hot", ou só de uma semana, ou de um vendedor
  • Emojis: 🔥 = Hot, 🌙 = Warm, ❄️ = Cold

🆘 Troubleshooting & Suporte

Problema: IA não está respondendo

Solução: Verifique se API keys estão corretas. Vá em Configuração → Chaves de API

Problema: IA está dando respostas ruins

Solução: IA espelha as instruções que você deu. Revise a seção "Configurar IA" e melhore as instruções

Problema: Mensagens estão atrasadas

Solução: Pode ser fila congestionada. Sistema processa 1000s/min. Se pico de tráfego, upgrade o plano.

Problema: Lead desapareceu do pipeline

Solução: Veja filtros - pode estar em "Lost" ou marcado como duplicado. Use busca global.

Problema: Banco de dados cheio

Solução: Limpe conversas antigas (>1 ano). Sistema avisa antes de ficar cheio.
