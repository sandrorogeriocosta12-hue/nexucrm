🔑 EXEMPLO DE FLUXO:

1. REGISTRAR USUÁRIO:
   curl -X POST http://localhost:8000/api/auth/register \
     -H "Content-Type: application/json" \
     -d '{"email":"test@example.com","password":"Pass123!","full_name":"Test User"}'

2. LOGIN:
   curl -X POST http://localhost:8000/api/auth/login \
     -H "Content-Type: application/json" \
     -d '{"username":"test@example.com","password":"Pass123!"}'
   
   → Resposta contém: access_token, refresh_token

3. USAR TOKEN PARA ACESSAR /me:
   curl -H "Authorization: Bearer <access_token>" \
     http://localhost:8000/api/auth/me

4. CRIAR LEAD:
   curl -X POST http://localhost:8000/api/leads \
     -H "Content-Type: application/json" \
     -H "Authorization: Bearer <access_token>" \
     -d '{
       "name":"João Silva",
       "email":"joao@example.com",
       "phone":"+5511999999999",
       "company":"Acme Corp",
       "source":"website"
     }'

5. QUERY KNOWLEDGE BASE:
   curl -X POST "http://localhost:8000/api/knowledge/query?company_id=acme&query=produto" \
     -H "Authorization: Bearer <access_token>"
