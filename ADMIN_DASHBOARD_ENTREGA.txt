🔐 SEGURANÇA
═══════════════════════════════════════════════════════════════════════════════

✅ JWT Authentication
   - Token com expiração 24 horas
   - Secret key configurável via RAILWAY
   - Validação em todas as rotas autorizadas

✅ Password Hashing (Bcrypt)
   - Impossível recuperar senha original
   - Salt de 12 rounds (secure)
   - Comparação segura em login

✅ Token Masking
   - Tokens nunca completamente expostos
   - Resposta: "EAA...***...GH7"
   - Apenas últimos 5 caracteres visíveis

✅ Input Validation
   - Email validation com Pydantic EmailStr
   - Senhas obrigatórias com comprimento mínimo
   - Tratamento de SQL injection
   - CORS restrictivo
