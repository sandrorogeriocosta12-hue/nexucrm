📋 PRÓXIMO PASSO:
─────────────────
Para obter os tokens, use este guia rápido:

🟦 WHATSAPP TOKEN:
   1. Vá para: https://developers.facebook.com/
   2. Selecione seu app
   3. Vá para: Settings > Basic
   4. Copie: App ID (exemplo: 123456789...)
   5. Cole no wizard

🟦 TELEGRAM TOKEN:
   1. Abra Telegram
   2. Procure: @BotFather
   3. Digite: /newbot
   4. Escolha nome + username
   5. @BotFather vai retornar: "123456:ABC-DEF123456..."
   6. Cole no wizard

🟦 SENDGRID API KEY:
   1. Vá para: https://sendgrid.com/
   2. Login
   3. Settings > API Keys
   4. CREATE API KEY (Full Access)
   5. Vai retornar: "SG.sk_live_abcdef123456..."
   6. Cole no wizard

🟦 META BUSINESS TOKEN:
   1. Vá para: https://facebook.com/business
   2. Settings > Business Settings
   3. System Users
   4. CREATE NEW > Generate Token
   5. Vai retornar um token longo
   6. Cole no wizard
