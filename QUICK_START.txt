❓ FAQ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

P: A página está em branco?
R: Verifique:
   1. Se localhost:8000 está online (console)
   2. Se JavaScript está habilitado
   3. Limpe cache (Ctrl+Shift+Delete)
   4. Recarregue (F5)

P: Botões não funcionam?
R: Verifique:
   1. Console do navegador (F12)
   2. Network tab para erros HTTP
   3. localStorage (DevTools)

P: Como voltar para o login?
R: Clique em 🚪 (logout) na sidebar

P: Como mudar de página?
R: Clique num ícone na sidebar ou use Ctrl+número

P: É possível adicionar mais páginas?
R: Sim! Adicione um novo <template> e um botão na sidebar

Para conectar com dados reais:

1. Abra DevTools (F12)
2. Vá para Console
3. Digite:

   fetch('/api/leads')
     .then(r => r.json())
     .then(console.log)

   // Você verá os dados da API!

4. Modifique o template HTML para renderizar dados reais
5. Use os dados na função renderPage()
