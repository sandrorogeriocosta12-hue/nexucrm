   🆘 SE ALGO DER ERRADO - ERROS COMUNS
════════════════════════════════════════════════════════════════════════════════

❌ ERRO: "401 Unauthorized - Invalid token"
   └─ Problema: Token está errado ou expirado
   └─ Solução:
      1. Copie o token COMPLETO (não deixe nada de fora)
      2. Se não funcionar, gere um novo em developers.facebook.com
      3. Espere 30 segundos antes de testar novamente


❌ ERRO: "Phone number ID inválido"
   └─ Problema: Copiar ID errado
   └─ Solução:
      1. Volte em business.facebook.com
      2. Vá para Settings → API Setup
      3. Procure novamente pelo Phone Number ID
      4. Cole CERTINHO, sem espaços ou símbolos


❌ ERRO: "Business Account ID inválido"
   └─ Problema: Id está incompleto
   └─ Solução:
      1. Copie o WAID inteiro (não deixe nada de fora)
      2. Se começar com "WAID", deixe assim mesmo


❌ ERRO: "Preencha os campos obrigatórios"
   └─ Problema: Deixou algum campo em branco
   └─ Solução:
      Certifique que preencheu:
      ✓ Phone Number ID (números)
      ✓ Business Account ID (pode começar com WAID)
      ✓ Access Token (longo, começa com EAA)
      ✓ Webhook Verify Token (qualquer coisa simples)


❌ ERRO: "Webhook verification failed"
   └─ Problema: Seu servidor não está acessível
   └─ Solução:
      1. Certifique que servidor está rodando
      2. Teste: curl http://localhost:8000
      3. Se não responde, rode: python app_server.py
