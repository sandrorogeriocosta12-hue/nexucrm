📖 Passo 1: Criar Aplicativo no Meta

  1. Acesse developers.facebook.com
  2. Crie um novo aplicativo do tipo "Business"
  3. Adicione o produto WhatsApp
  4. Configure sua conta comercial do WhatsApp

🔑 Passo 2: Gerar Tokens

  1. Em Settings → User Token, gere um novo token
  2. Copie o Access Token
  3. Vá para WhatsApp → API Setup
  4. Anote seu Phone Number ID

🔗 Passo 3: Configurar Webhook

  1. Em WhatsApp → Webhooks, clique em "Edit"
  2. Cole a URL acima em "Callback URL"
  3. Cole seu "Verify Token" abaixo
  4. Subscreva aos eventos: messages,message_template_status_update

💡 Teste Rápido

Após configurar, clique em "Testar Conexão" para validar tudo está funcionando corretamente.
