👉 Começar em 3 Passos

Passo 1: Setup Automático

python3 setup_integrations.py

Este script vai:

  • ✅ Verificar Docker + Python
  • ✅ Criar arquivo .env
  • ✅ Iniciar Evolution API
  • ✅ Criar database SQLite
  • ✅ Gerar relatório

Passo 2: Editar Credenciais

nano .env

Adicione suas credenciais Meta:

  • ✅ FACEBOOK_APP_ID (de https://developers.facebook.com)
  • ✅ FACEBOOK_APP_SECRET
  • ✅ INSTAGRAM_VERIFY_TOKEN (qualquer string aleatória)

Passo 3: Iniciar Servidor

python app_server.py

Abra navegador: http://localhost:8000/integrations-ui

  • ✅ Clique "Gerar QR Code" (WhatsApp)
  • ✅ Cole token Telegram (@BotFather)
  • ✅ Clique "Conectar Instagram" (OAuth)
  • ✅ 3/4 canais conectados! 🎉

🆘 Dúvidas Comuns

❓ QR Code não aparece

Solução: Verificar se Evolution API está rodando

curl http://localhost:3000/health

❓ Token Telegram inválido

Solução: Copiar novamente do @BotFather (verificar se tem ":")

❓ Instagram não conecta

Solução: Verificar FACEBOOK_APP_ID e FACEBOOK_APP_SECRET em .env

❓ Mensagens não chegam

Solução: Conferir se todos 3 canais mostram "✅ Conectado"

❓ Servidor não inicia

Solução: Porta 8000 pode estar em uso

lsof -ti:8000 | xargs kill -9

Mais detalhes em: INTEGRATION_GUIDE.md (seção Troubleshooting)
