🔗 ENDPOINTS PRONTOS
═════════════════════════════════════════════════════════════════════════════

┌─ LEADS ──────────────────────────────────────────────────────────────────────
│ GET    /api/leads                        Lista todos os leads
│ POST   /api/leads                        Cria novo lead
│ PUT    /api/leads/{id}                   Atualiza lead
│ DELETE /api/leads/{id}                   Deleta lead
│
├─ CAMPAIGNS ──────────────────────────────────────────────────────────────────
│ GET    /api/campaigns                    Lista campanhas
│ POST   /api/campaigns                    Cria campanha
│ PUT    /api/campaigns/{id}               Atualiza campanha
│
├─ CONTACTS ───────────────────────────────────────────────────────────────────
│ GET    /api/contacts                     Lista contatos
│ POST   /api/contacts                     Cria contato
│ PUT    /api/contacts/{id}                Atualiza contato
│ DELETE /api/contacts/{id}                Deleta contato (soft delete)
│
├─ MESSAGING ───────────────────────────────────────────────────────────────────
│ POST   /api/send-message                 Enviar via whatsapp/email/telegram/etc
│ POST   /api/webhooks/whatsapp            Receber mensagens WhatsApp
│ POST   /api/webhooks/telegram            Receber mensagens Telegram
│ POST   /api/webhooks/instagram           Receber mensagens Instagram
│
└─ MONITORING ──────────────────────────────────────────────────────────────────
  GET    /integrations/status              Ver status dos canais
  GET    /api/channels/status              Detalhe de cada canal

PASSO 1: VERIFICAR TUDO ESTÁ FUNCIONANDO

PASSO 2: EXPLORAR OS ENDPOINTS
─────────────────────────────────────────────────────────────────────────────
Acesse a documentação interativa:

    https://api.nexuscrm.tech/docs

Ou teste via curl:

    curl https://api.nexuscrm.tech/api/leads
    curl https://api.nexuscrm.tech/integrations/status


PASSO 3: CONFIGURAR CREDENCIAIS DOS CANAIS
─────────────────────────────────────────────────────────────────────────────
Para cada canal que quer ativar:

    A. WhatsApp:
       ├─ Ir em https://developers.facebook.com/
       ├─ Criar App Business
       ├─ Configurar WhatsApp API
       └─ Obter: WHATSAPP_ACCESS_TOKEN, WHATSAPP_PHONE_ID
    
    B. Telegram:
       ├─ No Telegram, procurar @BotFather
       ├─ Criar bot: /newbot
       └─ Copiar token para TELEGRAM_BOT_TOKEN
    
    C. Email:
       ├─ Ir em https://sendgrid.com/
       ├─ Criar conta gratuita
       └─ Obter API Key para SENDGRID_API_KEY
    
    D. Instagram/Facebook:
       ├─ Usar mesma conta que WhatsApp
       └─ Obter: META_BUSINESS_TOKEN, INSTAGRAM_BUSINESS_ID


PASSO 4: ADICIONAR CREDENCIAIS NO RAILWAY
─────────────────────────────────────────────────────────────────────────────
1. Acesse https://railway.app/
2. Acesse seu projeto Nexus CRM
3. Vá em Variables
4. Adicione:
   • WHATSAPP_ACCESS_TOKEN=...
   • WHATSAPP_PHONE_ID=...
   • TELEGRAM_BOT_TOKEN=...
   • SENDGRID_API_KEY=...
   • META_BUSINESS_TOKEN=...
   
5. Deploy (automático)


PASSO 5: TESTAR OS CANAIS
─────────────────────────────────────────────────────────────────────────────
Enviar mensagem de teste:

    curl -X POST https://api.nexuscrm.tech/api/send-message \
      -H 'Content-Type: application/json' \
      -d '{
        "channel": "whatsapp",
        "destination": "5511999999999",
        "text": "Teste do Nexus CRM"
      }'

Verificar status:

    curl https://api.nexuscrm.tech/integrations/status

Esperado: "configured": true para cada canal com credenciais


PASSO 6: CONFIGURAR WEBHOOKS (Opcional mas Recomendado)
─────────────────────────────────────────────────────────────────────────────
Para receber mensagens em tempo real:

    WhatsApp:
    ├─ Dashboard Facebook Developers
    ├─ Webhooks
    └─ Callback URL: https://api.nexuscrm.tech/api/webhooks/whatsapp
    
    Telegram:
    ├─ No BotFather use: /setwebhook
    └─ URL: https://api.nexuscrm.tech/api/webhooks/telegram
    
    Instagram:
    └─ URL: https://api.nexuscrm.tech/api/webhooks/instagram

✨ EXEMPLOS DE USO
═════════════════════════════════════════════════════════════════════════════

Criar um lead e enviar WhatsApp:

    # 1. Criar lead
    curl -X POST https://api.nexuscrm.tech/api/leads \
      -H 'Content-Type: application/json' \
      -d '{
        "user_id": "minha-empresa",
        "name": "João Silva",
        "phone": "5511987654321",
        "value": 25000,
        "status": "novo"
      }'
    
    # 2. Enviar mensagem
    curl -X POST https://api.nexuscrm.tech/api/send-message \
      -H 'Content-Type: application/json' \
      -d '{
        "channel": "whatsapp",
        "destination": "5511987654321",
        "text": "Olá João! Como va?"
      }'
    
    # 3. Registrar campanha
    curl -X POST https://api.nexuscrm.tech/api/campaigns \
      -H 'Content-Type: application/json' \
      -d '{
        "user_id": "minha-empresa",
        "name": "Follow-up Q2",
        "channel": "whatsapp",
        "budget": 10000
      }'
