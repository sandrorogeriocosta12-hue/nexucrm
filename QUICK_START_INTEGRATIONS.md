## 📱 WhatsApp - "Um Clique" com QR Code

```
1. Clique em "Gerar QR Code"
2. Aponte seu celular (câmera)
3. Escaneie com qualquer app de câmera
4. Pronto! WhatsApp conectado ✓
```

**Quanto tempo?** ~30 segundos  
**Necessário:** Nada! Apenas câmera do celular

---

## 🤖 Telegram - Copiar e Colar

```
1. Abra Telegram no seu celular
2. Procure por: @BotFather
3. Envie comando: /newbot
4. Siga as orientações (nome + username)
5. Copie o TOKEN recebido
6. Volte aqui e cole no campo
7. Clique "✓"
```

**Quanto tempo?** ~1 minuto  
**Token exemplo:** `123456:ABCdefGHIjklMNOpqrsTUVwxyz-1234567890`

## 💬 O Sistema Faz...

Quando uma mensagem chega:

```
Cliente → WhatsApp/Telegram/Instagram
        ↓
Nexus recebe em tempo real
        ↓
IA analisa a mensagem (score 0-100)
        ↓
Se for LEAD (score > 80%):
  → Sistema responde automaticamente
  → "Obrigado! Vamos analisar sua solicitação"
        ↓
Dashboard mostra todas as mensagens
        ↓
Você pode responder/categorizar
```

## 🆘 Algo Deu Errado?

### "QR Code não aparece"
→ Abra console (F12) e procure por erros vermelhos

### "Token Telegram inválido"
→ Copie novamente, certifique do BotFather

### "Instagram não conecta"
→ Faça logout e tente novamente com account Business

### "Mensagens não chegam"
→ Verifique se todos 3 canais estão "✅ Conectado"
