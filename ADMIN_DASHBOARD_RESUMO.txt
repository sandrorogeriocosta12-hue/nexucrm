ENDPOINTS PRINCIPAIS
───────────────────────────────────────────────────────────────────────────────

POST   /api/admin/register     ← Novo cliente se registra
POST   /api/admin/login        ← Cliente faz login
POST   /api/admin/tokens       ← Salva tokens dos canais
GET    /api/admin/tokens       ← Vê tokens (mascarados)
DELETE /api/admin/tokens/{ch}  ← Deleta token de um canal
GET    /api/admin/status       ← Status em tempo real
GET    /api/admin/user         ← Perfil do usuário
GET    /login                  ← Página de login
GET    /dashboard              ← Dashboard do cliente

COMO USAR
───────────────────────────────────────────────────────────────────────────────

1. INICIAR SERVIDOR:
   $ python3 app_server.py

2. ACESSAR NO NAVEGADOR:
   http://localhost:8000/login

3. CRIAR CONTA:
   Email:    admin@empresa.com
   Senha:    Sua senha segura
   Nome:     Seu Nome
   Empresa:  Sua Empresa

4. ADICIONAR TOKENS:
   - Dashboard → Aba "Tokens"
   - Preencha tokens do WhatsApp, Telegram, etc
   - Clique "Salvar Tokens"
   - Pronto! Sistema configura automaticamente

5. MONITORAR:
   - Dashboard → Aba "Status"
   - Vê quais canais estão ✅ Ativo ou ❌ Inativo
