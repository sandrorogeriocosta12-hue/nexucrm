🔐 SEGURANÇA & AUTENTICAÇÃO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   JWT Tokens
   ├─ Access Token
   │  ├─ Tempo: 30 minutos
   │  ├─ Header: Authorization: Bearer <token>
   │  └─ Claims: user_id, email, exp
   │
   └─ Refresh Token
      ├─ Tempo: 7 dias
      └─ Renovação automática de access tokens

   Password
   ├─ Hashing: Argon2-cffi
   ├─ Salt: Incluído automaticamente
   └─ Comparação: Segura contra timing attacks

   localStorage
   ├─ Armazena: vexus_token (access token)
   ├─ Scope: Origin específico (localhost:8000)
   ├─ Session: Persiste até logout
   └─ Logout: removeItem('vexus_token')

   Endpoints Protegidos
   ├─ GET  /api/leads       → Requer auth
   ├─ GET  /api/campaigns   → Requer auth
   ├─ POST /api/auth/login  → Sem auth (público)
   └─ POST /api/auth/register → Sem auth (público)
